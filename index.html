COVID-19 IN MALAYSIA

Visualized | Last Updated: 10/05/2022

25 January 2020 marked the first day of COVID-19 pandemic in Malaysia. Since 5 May 2020, Ministry of Heath Malaysia (MOH) has been collecting data of COVID-19 in Malaysia. Here is some visualisation on the collected data.

Active COVID-19 Cases In Malaysia

Here is a map showing the active cases in Malaysia by selected date through using the slider filter below.

Deaths by COVID-19 In Malaysia

This chart is showing the number of deaths by each state in the selected interval. There is a timeline to select the interval.
Black (>100), purple (1-100) and green (0).

COVID-19 Vaccinations In Malaysia

This chart is showing the number of vaccinations distributed each day. We can see that 1st Dose and 2nd Dose have similar pattern as most of the population had received their vaccinations in a 3 weeks interval.
